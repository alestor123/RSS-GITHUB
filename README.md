# Documentation

# Quick use

``npx rss-github``

# Installation

``npm i rss-github -g ``

# Usage

``rss-github -p 3000 ``

# Example
``rss-github -p 3000  -t token -f ./lol.log -l 50  ``

# Env

## Creating Env
``touch .env``

# Env Example

## Usage
```
TOKEN=<Token> 
LIMIT=<Limit number>
LOG=<true or false >
LOGPATH=<Path to the log file>
PORT=<Port Number>-p 3000  -t token -f ./lol.log -l 50 
```
## Example

```
TOKEN=blahblahbal 
LIMIT=5
LOG=true
LOGPATH=./lolo.log
PORT=3000
```
